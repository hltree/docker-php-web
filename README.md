# Docker + PHP + Web

DockerでPHP環境を作成するサンプル集です。

構築の際の参考にしてください。

## PHP Version

- 7.4

## Environment

- Mac OS Big Sur 11.5.2（確認済） 
- Docker version 20.10.8, build 3967b7d

## Usage

1. gitをclone
```
Use SSL
$ git clone git@github.com:hltree/docker-php-web.git

Use HTTPS
$ git clone https://github.com/hltree/docker-php-web.git
```

2. 利用元を利用先にコピーする
```
cp -RT ${from} ${to}
```
